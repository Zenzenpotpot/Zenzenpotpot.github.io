📊 Automated Grading System

VB.NET desktop application for grade management and reporting

📌 Overview

A complete desktop application built with VB.NET that allows teachers to manage student grades, compute averages, generate report cards, and export data to Excel. It eliminates manual calculation errors and saves hours of administrative work.

⚠️ Problem it Solves

Teachers often spend hours manually calculating grades and preparing report cards. This system automates the entire process, providing instant results and formatted reports, reducing workload and human error.

🛠️ Tech Stack

VB.NET SQL Server Crystal Reports Windows Forms

✨ Key Features

  • Add, edit, and delete student records
  • Input scores per subject with flexible grading periods
  • Automatic grade computation (weighted average, transmutation)
  • Generate printable report cards (PDF)
  • Export data to Excel for further analysis
  • User authentication for teachers and admin

💻 Code Snippet

Example function that computes the average of scores:

Public Function ComputeAverage(scores As List(Of Double)) As Double
    If scores.Count = 0 Then Return 0
    Return scores.Sum() / scores.Count
End Function

🧩 Challenges & Solutions

Challenge: Integrating Crystal Reports with VB.NET caused version compatibility issues.
Solution: Installed the correct Crystal Reports runtime and referenced the proper DLLs to ensure seamless report generation.

Challenge: Handling large datasets efficiently.
Solution: Used stored procedures and optimized SQL queries to minimize load times.
