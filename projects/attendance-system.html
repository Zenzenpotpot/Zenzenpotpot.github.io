🔐 Smart Attendance System

Arduino + RFID + Fingerprint + SMS Integration

📌 Overview

A hardware‑integrated attendance system that uses RFID and fingerprint sensors to log student attendance. Upon successful verification, a VB.NET application records the data and sends an SMS notification to parents or administrators.

⚠️ Problem it Solves

Manual attendance is time‑consuming and prone to proxy attendance. This system ensures that only the authorized student can mark attendance, and parents receive real‑time updates, improving accountability and transparency.

🛠️ Tech Stack

Arduino Uno RFID RC522 Fingerprint Sensor (AS608) GSM SIM800L VB.NET Serial Communication MySQL Server

✨ Key Features

  • Dual‑factor authentication: RFID + fingerprint
  • Real‑time SMS alert via GSM module
  • VB.NET interface displays logs and allows admin control
  • Automatic generation of daily/monthly attendance reports
  • Offline capability: attendance stored locally on Arduino if PC disconnected

🧩 Challenges & Solutions

Challenge: Integrating the fingerprint sensor with Arduino required precise timing; sensor communication sometimes blocked the serial port.
Solution: Implemented a state machine to manage sensor communication without halting the main loop.

Challenge: GSM module occasionally failed to send SMS.
Solution: Added retry logic and a queue system to ensure delivery; also used AT command error checking.

Challenge: Synchronizing the Arduino clock with PC time for accurate logging.
Solution: Sent time sync commands from VB.NET to Arduino on startup.

📖 What I Learned

  • Hardware‑software integration and serial communication protocols
  • Designing fail‑safe systems for real‑time operations
  • Power management for embedded devices
  • Debugging hardware issues with oscilloscopes and logic analyzers
